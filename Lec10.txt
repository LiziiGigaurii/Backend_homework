1)რა განსხვავებაა let, const შორის?
let შეგვიძლია შევცვალოთ და მივანიჭოთ სხვა მნიშვნელობა
const — ერთხელ ანიჭებ და მეტს ვერ შეცვლი


2)რა განსხვავებაა primitive და nonprimitive type-ებს შორის?
primitive არის ისეთი მარტივი სტრუქტურები, რომლებსაც ვერანაირად ვერ შევცვლით (string, undefined, number , boolean, null)
nonprimitive პირიქით რთული სტრუქტურაა, რომლის შეცვლის უფლებაც გვაქვს (Object, function, array)


3)რა განსხვავებაა == და === ოპერატორებს შორის?
== ადარებს მოცემული სტრუქტურების მხოლოდ მნიშვნელობას, ტიპს კი უგულებელყოფს, 
ხოლო === ადარებს მნიშვნელობასაც და ტიპსაც


4)რა ტიპის არის null
null არის "არაფერი", ცარიელი მნიშვნელობა. typeof null-ს თუ დავლოგავთ მივიღებთ "object"-ს,
რაც ჯავასკრიპტის ბაგია. მისი გამოსწორება კი უფრო მეტ პრობლემას გამოიწვევს.


5)რა განსხვავებაა funtion და = () => {} შორის
მთავარი განსხვავება this-შია. function არის ფუნქცია, რომელსაც თავისი this აქვს.
() => {}-ს კი საკუთარი this არ აქვს, და მას გარე კონტექსტიდან იღებს


6)რას ნიშნავს callback function?
ეს არის ფუნქცია, რომელსაც არგუმენტად სხვა ფუნქციას გადავცემთ და შემდგომ ვიძახებთ.


7)რა განსხვავებაა map, filter და reduce-ს შორის?
map - ლისთის თითოეულ ელემენტს გარდაქმნის და გვიბრუნებს ახალ ლისთს
filter - სახელიდან გამომდინარე ფილტრავს ლისთს და map-ის მსგავსად ესეც ახალ ლისთს აბრუნებს
reduce - შედგება total-სგან და accumulator-სგან, რომელიც გვეხმარება მონაცემების დაჯამებაში.ის ერთ მნიშვნელობას აბრუნებს


8)რა არის Promise?
promise არის ისეთი პირობა რომელიც საბოლოოდ აუცილებლად დააბრუნებს ან rejected ან fulfilled


9)რა განსხვავებაა Promise.all, Promise.race და Promise.any-ს შორის?
Promise.all - აბრუნებს ყველა პასუხს, თუმცა თუ ერთი rejected იქნება მაშინ ყველა rejected გახდება
Promise.race - აბრუნებს ყველაზე სწრაფ პირველ პასუხს
Promise.any - აბრუნებს პირველ fulfilled პირობას


10)რას ნიშნავს async/await და რატომ ვიყენებთ?
ვიყენებთ ძირითადად fetch(API)-ს დროს. ამ დროს კოდი გამოიყურება სინქრონულად, მაგრამ ასინქრონულად მუშაობს


11)რა განსხვავებაა class-სა და object-ს შორის?
class არის ერთგვარი ბლუპრინტი, ნიმუში, რომელსაც აღვწერთ სხვადასხვა მეთოდებითა და თვისებებით, ხოლო
object არის უკვე class-დან new-ით შექმნილი კონკრეტული მაგალითი


12)რას აკეთებს constructor class-ში?
constructor არის მეთოდი, რომელიც გამოიყენება class-ის შიგნით მის აღსაწერად და ინიციალიზაციისთვის
იგი new-ით შექმნისთანავე გამოიძახება და ობიექტს საწყის მნიშვნელობებს ანიჭებს


13)რა განსხვავებაა მემორი ჰიპსა და მემორი სტექკს შორის
მემორი სტაკი ინახავს პრიმიტიულ მნიშვნელობებს და ფუნქციების გამოძახებებს, მცირე ზომისაა და სწრაფია, ხოლო
მემორი ჰიპი ინახავს ობიექტებს და ლისთებს


14) რა არის ქოლსტეკი
ეს არის სია, სადაც ჯავასკრიპტი თვლის რომელ ფუნქციაშია ამჟამად


15) რა არის ივენთ ლუპი
ელოდება ქოლსტეკის გათავისუფლდეს, შემდეგ ქოლბექ Queue-დან ასინქრონულ ამოცანებს უშვებს შიგნით. 
სწორედ ის აძლევს ჯავასკრიპტს ასინქრონულობის საშუალებას, მიუხედავად იმისა რომ single-threaded არის.
